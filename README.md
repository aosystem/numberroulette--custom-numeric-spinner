# Number Roulette: Random Draw Tool 🎯🔢

A professional and highly customizable **Digital Number Roulette** built with **Flutter**. Designed for fairness and engagement, this utility allows users to create custom roulette boards for raffles, education, and quick decision-making.

## 🌟 Key Features
- **Custom Range Selection:** Set any numeric range (0 or greater) to fit your specific needs (e.g., 1–100, 101–150).
- **Flexible Layouts:** Choose between Ascending, Descending, or **Random (Shuffled)** order for the roulette board.
- **Voice Announcement:** Results are read aloud using Text-to-Speech (TTS) for better engagement during events.
- **Global Accessibility:** Fully localized in **50+ languages**, supporting users across the globe.

## 💡 Best For...
- **Events & Raffles:** A reliable digital alternative to physical drawing machines.
- **Classrooms:** Randomly selecting students or managing group activities.
- **Decision Making:** Making quick, unbiased choices in a fun and visual way.

## 🛠 Technical Overview
- **Framework:** Flutter
- **Language:** Dart
- **Logic:** Dynamic UI generation for the roulette board based on custom ranges and integrated Text-to-Speech (TTS) for result announcements.

*Note: This repository contains the core logic and source code. Assets (images, sound effects, and localization files) are excluded to protect intellectual property.*
